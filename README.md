# VC_Blog Content Source

This repo is used as the public content source for Vincent's website.

## Files

- `content.json`: Intro, projects, contact text, links, and image paths.
- `blogs.json`: Blog metadata and markdown URLs/paths.
- `img/`: Images referenced by JSON.

## Notes

- Relative paths in JSON are resolved against:
  `https://raw.githubusercontent.com/Galile-Vincent/VC_Blog/main/...`
- You can use full URLs too.
